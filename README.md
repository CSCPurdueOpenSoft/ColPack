CMake Build Instructions
=========================
1. Get CMake (https://cmake.org/)
2. In the 'ColPack' directory run:
	mkdir build <br />
	cd build <br />
	cmake .. <br />
	make <br />
	make install <br />

This installs the dynamic library into 'ColPack'/ColPack/lib64  
and the header files into 'ColPack'/ColPack/include/ColPack.
